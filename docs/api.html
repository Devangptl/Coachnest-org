All API routes are served under /api/. Requests and responses use JSON unless noted (file uploads use multipart/form-data, certificate downloads return a PDF stream).

Auth Legend

BadgeMeaning
PublicNo authentication required
Any RoleRequires a valid session (any role)
StudentSTUDENT role only
Ins / AdminINSTRUCTOR or ADMIN role
InstructorINSTRUCTOR role only
AdminADMIN role only

⚠️ Error Codes

StatusMeaning
400Bad request — missing or invalid fields
401Unauthorized — no valid session
402Payment required — course requires purchase
403Forbidden — insufficient role or not resource owner
404Not found
409Conflict — duplicate resource (email, slug, etc.)
413Payload too large — file exceeds size limit
415Unsupported media type — not an image
422Unprocessable — business rule violation
500Internal server error

🔐 Authentication

POST /api/auth/signup Public

Register a new user account.

Request body

{ "name": "Jane Doe", "email": "jane@example.com", "password": "Password123!", "role": "STUDENT" }

role is optional — defaults to STUDENT. Pass "INSTRUCTOR" to register as an instructor.

Response 201

{ "message": "Account created.", "userId": "uuid" }

Errors

400 missing fields  ·  409 email already exists

POST /api/auth/login Public

Log in and set the HttpOnly session cookie.

Request body

{ "email": "jane@example.com", "password": "Password123!" }

Response 200

{ "message": "Logged in.", "role": "STUDENT" }
POST /api/auth/logout Public

Clear the session cookie.

Response 200

{ "message": "Logged out." }
GET /api/auth/me Any Role

Get the currently authenticated user.

Response 200

{ "user": { "userId": "uuid", "email": "jane@example.com", "name": "Jane Doe", "role": "STUDENT", "avatar": "https://..." } }
POST /api/auth/forgot-password Public

Send a password reset email via Supabase.

Request body

{ "email": "jane@example.com" }

Response 200

{ "message": "If that email exists, a reset link was sent." }

📚 Courses

GET /api/courses Public

List all published courses with ratings and enrollment counts.

Response 200

{ "courses": [{ "id": "uuid", "title": "Intro to Python", "price": 49.99, "isFree": false, "level": "BEGINNER", "avgRating": 4.7, "createdBy": { "name": "Alice" }, "_count": { "enrollments": 120 } }] }
POST /api/courses Ins / Admin

Create a new course (defaults to DRAFT status).

Request body

{
  "title": "Advanced React",
  "description": "...",
  "price": 79.99,
  "isFree": false,
  "level": "ADVANCED",
  "language": "English",
  "categoryId": "uuid",
  "instructorRevenuePercent": 75,
  "tagNames": ["react", "typescript"]
}

Response 201

{ "course": { "id": "uuid", ... } }
GET /api/courses/[id] Public

Get a single course with sections and lessons.

Response 200

{ "course": { "id": "uuid", "sections": [{ "lessons": [...] }], "_count": { "enrollments": 120 } } }
PATCH /api/courses/[id] Ins / Admin

Update a course. Only the owner or an admin can update.

Request body

{ "title": "Updated Title", "price": 59.99, "published": true }
DELETE /api/courses/[id] Ins / Admin

Response 200

{ "message": "Course deleted." }

ParamDescription

📝 Quizzes

GET/api/quizzes?lessonId=uuidPublic

Get a quiz for a lesson. isCorrect is stripped for students (only admins/instructors see correct answers).

Response 200

{ "quiz": { "id": "uuid", "passMark": 70, "timeLimit": 600, "questions": [{ "id": "uuid", "text": "...", "options": [...] }] } }
POST/api/quizzesIns / Admin

Create a quiz with questions and options for a lesson.

Request body

{
  "lessonId": "uuid", "title": "Module Quiz",
  "passMark": 70, "timeLimit": 600,
  "questions": [{
    "text": "What does JSX stand for?", "order": 1, "points": 1,
    "options": [
      { "text": "JavaScript XML", "isCorrect": true },
      { "text": "Java Syntax Extension", "isCorrect": false }
    ]
  }]
}
POST/api/quizzes/[id]/attemptAny Role

Submit answers for a quiz attempt. Returns score, pass/fail, and XP gained.

Request body

{ "answers": { "questionId": "selectedOptionId" }, "timeTaken": 245 }

Response 200

{ "score": 85, "passed": true, "passMark": 70, "xpGained": 100, "feedback": [{ "questionId": "uuid", "correct": true }] }
GET/api/quiz-historyAny Role

Get the current user's quiz attempt history grouped by quiz.

Response 200

{ "data": [{ "quizId": "uuid", "bestScore": 85, "hasPassed": true, "attemptCount": 2 }] }

🏅 Certificates

GET/api/certificates/[courseId]Any Role

Generate and download a PDF certificate. All lessons in the course must be completed.

Query params

Response

PDF binary stream with Content-Disposition: attachment; filename="certificate.pdf"

Errors

403 course not completed  ·  404 enrollment not found

💳 Payments

POST/api/payments/create-payment-intentAny Role

Create a Stripe Checkout session for a course purchase.

Request body

{ "courseId": "uuid", "couponCode": "SAVE20", "referralCode": "REF-ABC123" }

Response 200

{ "clientSecret": "cs_test_...", "orderId": "uuid", "amount": 3999, "courseName": "Advanced React" }

amount is in smallest currency unit (cents for USD).

POST/api/payments/verifyAny Role

Confirm a completed Stripe checkout and finalize enrollment.

Request body

{ "sessionId": "cs_test_..." }

Response 200

{ "success": true, "enrollmentId": "uuid", "courseId": "uuid" }
POST/api/payments/webhookStripe Signature

Stripe webhook endpoint. Verifies Stripe-Signature header against STRIPE_WEBHOOK_SECRET. Handles async events (late payments, failures).

Response 200

{ "received": true }

List saved payment methods.

POST/api/billing/payment-methodsAny Role

Add a payment method via SetupIntent.

DELETE/api/billing/payment-methods/[id]Owner only

Remove a saved payment method.

📤 Upload & Media

POST/api/uploadIns / Admin

Upload an image to Cloudinary. Content-Type: multipart/form-data. Max 1 MB per file, 100 MB per user.

Fields

file (required)  ·  folder: courses | blogs | avatars | misc

Response 201

{ "url": "https://res.cloudinary.com/...", "publicId": "courses/abc", "width": 1280, "height": 720, "format": "jpg" }

Errors

413 file too large  ·  415 not an image  ·  422 user storage limit exceeded

POST/api/upload/avatarAny Role

Upload and set a profile avatar. Max 1 MB. Content-Type: multipart/form-data.

GET/api/mediaIns / Admin

List uploaded media assets. Cursor-based pagination.

Query params

folder?  ·  cursor?  ·  limit? (default 40, max 100)

DELETE/api/media/[id]Owner / Admin

Delete a media asset from Cloudinary and the database.
